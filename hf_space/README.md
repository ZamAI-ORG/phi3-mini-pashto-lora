# ZamAI Phi-3 Mini Pashto

Pashto instruction-tuned language model based on Microsoft Phi-3 Mini.

د پښتو ژبې لپاره د هوښیارۍ ماډل

## Model

This space uses the [tasal9/ZamZeerak-Phi3-Pashto](https://huggingface.co/tasal9/ZamZeerak-Phi3-Pashto) model.

## Features

- Bilingual interface (Pashto/English)
- Safety filtering on inputs and outputs
- Configurable generation parameters
- Example prompts for quick testing

## Usage

Simply enter your question or instruction in Pashto or English, and the model will generate a response.

## Limitations

This model is for research and educational purposes. Responses may not always be accurate.
